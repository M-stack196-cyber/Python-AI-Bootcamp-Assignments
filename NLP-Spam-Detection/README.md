# Spam Email / SMS Detection (NLP)

## Project Overview
This project detects whether a given message is Spam or Ham using Natural Language Processing techniques.

## Dataset
SMS Spam Collection

## Model
- TF-IDF for feature extraction
- Naive Bayes / Logistic Regression for classification

## Tools & Libraries
- Python
- Scikit-Learn
- Pandas
- NumPy

## Goal
To accurately classify messages into Spam or Ham categories.
