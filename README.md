# Python & AI BootCamp Assignments

## Overview
This repository contains three AI projects completed as part of the Python & AI BootCamp assignment.

## Projects
1. **CNN - Cat vs Dog Image Classification**
   - Dataset: Kaggle Cats vs Dogs
   - Model: Convolutional Neural Network
   - Goal: Binary image classification

2. **Transfer Learning - Flower Species Classification**
   - Dataset: TensorFlow Flowers Dataset
   - Model: MobileNetV2 (Pretrained)
   - Goal: 5-class flower classification

3. **NLP - Spam Email/SMS Detection**
   - Dataset: SMS Spam Collection
   - Model: TF-IDF + Naive Bayes/Logistic Regression
   - Goal: Spam vs Ham classification

## Tools Used
- Python
- TensorFlow/Keras
- Scikit-Learn
- Pandas, NumPy, Matplotlib
