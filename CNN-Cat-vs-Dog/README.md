# Cat vs Dog Image Classifier (CNN)

## Project Overview
This project uses a Convolutional Neural Network (CNN) to classify images
as either Cat or Dog.

## Dataset
Kaggle Cats vs Dogs Dataset

## Model
- Convolutional Neural Network (CNN)
- Binary Image Classification

## Tools & Libraries
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## Goal
To build a model that can accurately distinguish between cat and dog images.
